## Day 8 - Linux Terminal Hacks

### Shortcuts learned:
- Ctrl+R → search command history
- Ctrl+A → jump to line start
- Ctrl+E → jump to line end
- Ctrl+U → delete entire line
- Tab → autocomplete

### Tools learned:
- tmux → split terminal into multiple panels
- alias → create command shortcuts
- history → see all past commands

### Aliases I created:
- ll = ls -la
- cls = clear

### Mini challenge:

### Key learning:
- Terminal shortcuts save huge time daily
- tmux lets you run multiple things at once
- Security engineers use tmux to monitor
  multiple servers simultaneously
