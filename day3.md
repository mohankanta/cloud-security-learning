## Day 3 - Linux Package Management

### Commands learned:
- sudo apt update → update package list
- sudo apt upgrade → upgrade pakage list
- sudo apt install → install a package
- sudo apt remove → remove a package
- sudo apt autoremove → clean unused packages
- dpkg --list → see all installed packages
- pip install → install python packages
