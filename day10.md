## Day 10 - What is a Network

### Key concepts learned:
- Network = 2 or more devices connected
- LAN = Local Area Network (home/college)
- WAN = Wide Area Network (internet)
- Router = connects different networks
- Switch = connects devices in same network
- Packet = small chunk of data sent over network
- IP Address = unique address of every device
- MAC Address = physical address of network card

### Commands learned:
- ip addr → see my IP address
- ip route → see router/gateway IP
- ping → test connectivity
- traceroute → see path of data
- ss -tulnp → see active connections

### Key learning:
- Every device on a network has unique IP
- Data travels in packets through routers
- Traceroute shows exact path data takes
- This is foundation of cloud networking
